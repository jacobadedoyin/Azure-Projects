Azure AZ-900 Projects

Basic hands-on projects aligned with Microsoft Azure Fundamentals (AZ-900).
Purpose: learn core cloud concepts, Azure services, pricing, and governance through simple, practical examples.

What this repo covers
* Cloud concepts: IaaS, PaaS, SaaS
* Core Azure services: compute, storage, networking
* Azure pricing, cost management, and SLAs
* Security, governance, and compliance basics

Goal
* Build a solid foundation before progressing to AZ-104 and AZ-305.
