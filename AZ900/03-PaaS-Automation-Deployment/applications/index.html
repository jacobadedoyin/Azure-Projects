Project 03: Static Web App Deployment

This page is hosted using Azure Static Web Apps. It demonstrates the efficiency of Platform as a Service (PaaS).

Why PaaS?

  • Zero Server Management: No OS updates or IIS/Apache configuration.
  • Global Distribution: Automatically deployed to Azure's global content delivery network.
  • Auto-Scaling: Handles traffic spikes without manual intervention.
  • CI/CD Integration: Deployed automatically via GitHub Actions.

"In an IaaS model (like a VM), I manage the server. In this PaaS model, I only manage the content."
