Project 01: Azure Resource Groups

Objective:
Understand how Azure subscriptions and resource groups organise cloud resources

Steps:
Created an Azure subscription
Created a Resource Group in a chosen region
Deployed a basic Azure Storage Account inside the Resource Group
Reviewed resources and metadata in Azure Portal
Deleted the Resource Group to remove all resources and avoid costs

AZ-900 Concepts Covered:
Subscriptions vs Resource Groups
Azure regions
Resource lifecycle management
Cost control and cleanup

Outcome:
Build a clear understanding of how Azure structures and manages resources
